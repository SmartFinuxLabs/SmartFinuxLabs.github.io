Analysis: Architecture Flow

1

Discovery & Resolution

Client queries the Wallet Address Server using a Payment Pointer (e.g., $wallet.com/alice) to resolve endpoints for Authorization and Resources.

2

Auth Server & IdP Consent

Client requests a GNAP grant from the Authorization Server, which seamlessly delegates to the Identity Provider (IdP) for cryptographic user consent validation.

3

Resource Server Access

Armed with an access token, the Client calls the Resource Server (the ASE's API gateway). This component initiates incoming or outgoing payments and bridges to the ILP node.

4

ILP Accounting Ledger

As ILP packets route through the Resource Server, the internal Accounting Service Entity continuously increments or decrements the peer's Accounts Payable/Receivable balances.

5

Automated Settlement

The Settlement Engine monitors the Accounting thresholds. Upon breach, it executes a deterministic, irrevocable value transfer on the underlying Core Ledger.
